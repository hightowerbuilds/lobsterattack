# Daily Summary Guide

Update the daily summary file for today. If a file for today does not exist, create a new one.

## File naming

Each daily summary file should be named using the format: `MM-DD-YYYY.md`

For example: `04-05-2026.md`

## Instructions

1. Check if a file for today's date already exists in this directory.
2. If it exists, update it with the latest summary.
3. If it does not exist, create a new file with today's date as the filename.
